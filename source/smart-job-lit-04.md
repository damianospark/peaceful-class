# 4강 | 생성형 AI 활용: 사업 기획 실습

## 오리엔테이션

Duration: 15

### 개요

이 강의에서는 **생성형 AI를 활용한 실무 기획 역량**을 키웁니다. 2025년 취업 시장에서 생성형 AI 활용 능력은 필수 역량이 되었습니다. 단순히 도구를 사용하는 것을 넘어, **프롬프트 엔지니어링을 통해 나만의 경쟁력**을 만들어가는 방법을 배웁니다.

### 🎯 학습 목표

1. ✅ 생성형 AI의 개념과 2025년 트렌드 이해
2. ✅ 프롬프트 엔지니어링 기본 원리 습득
3. ✅ 사업 기획서 자동생성 및 검증 실습
4. ✅ AI 출력물을 나만의 스타일로 가공하는 능력 배양

<aside class="info">
대학생의 92%가 AI 도구를 학습에 사용 중입니다. 하지만 단순 활용을 넘어 '전략적 활용'이 취업 경쟁력의 핵심입니다.
</aside>

### 실습 시나리오

**사업 아이디어**: 생성형 AI 기반의 저비용 매장 음악 제공 서비스

이번 강의에서는 위 사업 아이디어를 생성형 AI를 활용하여 구체화하고 기획서를 작성합니다.

| 단계 | 활동 | 생성 산출물 | 도구 |
|------|------|-------------|------|
| **1. 아이디어 발굴** | 시장 분석 및 문제점 도출 | 시장 분석 문서 | ChatGPT/Gemini |
| **2. 솔루션 설계** | 서비스 핵심 기능 정의 | 기능 명세서 | ChatGPT/Gemini |
| **3. 기획서 작성** | 사업계획서 초안 생성 | 사업계획서 (1-2페이지) | ChatGPT/Gemini |
| **4. 검증 및 개선** | 프롬프트 개선 및 보완 | 최종 기획서 | ChatGPT/Gemini |

### 준비사항

1. **필수 도구**: ChatGPT (무료) 또는 Google Gemini (무료) 계정
2. **권장 도구**: Claude, Grok 등 (선택사항)
3. **문서 도구**: Google Docs 또는 Notion (결과물 정리용)

## 생성형 AI 이해하기

Duration: 20

### 생성형 AI란?

생성형 AI(Generative AI)는 **텍스트, 이미지, 코드 등을 자동으로 생성**하는 인공지능입니다. 2025년 현재 가장 주목받는 기술 트렌드 중 하나입니다.

### 주요 생성형 AI 도구

#### 1. **ChatGPT** (OpenAI)
- 가장 널리 사용되는 대화형 AI
- 무료 버전(GPT-3.5)과 유료 버전(GPT-4) 제공
- 강점: 자연스러운 대화, 다양한 작업 지원

#### 2. **Google Gemini**
- Google의 차세대 AI 모델
- Google 서비스와 통합
- 강점: 검색 결과 연동, 최신 정보 반영

#### 3. **Claude** (Anthropic)
- 윤리적 AI로 평가받는 모델
- 강점: 긴 문서 처리, 세밀한 분석

#### 4. **GitHub Copilot**
- 코드 생성 특화 AI
- 강점: 프로그래밍 작업 지원

### 프롬프트 엔지니어링 기본

**프롬프트**는 AI에게 주는 지시문입니다. 좋은 프롬프트가 좋은 결과를 만듭니다.

#### ❌ 나쁜 프롬프트 예시

```
사업 계획서 써줘
```

**문제점**: 너무 모호함, 구체적인 정보 부족

#### ✅ 좋은 프롬프트 예시

```
나는 대학생 창업을 준비 중입니다. 
"생성형 AI 기반의 저비용 매장 음악 제공 서비스"에 대한 
사업계획서를 작성하려고 합니다.

다음 항목을 포함해주세요:
1. 서비스 개요 (3줄)
2. 타겟 고객 (구체적으로)
3. 핵심 기능 3가지
4. 경쟁 우위
5. 수익 모델

각 항목은 명확하고 간결하게 작성해주세요.
```

**장점**: 역할, 주제, 원하는 결과물이 명확함

### 프롬프트 작성 4원칙

1. ✅ **역할 지정**: "당신은 창업 컨설턴트입니다"
2. ✅ **맥락 제공**: 배경, 목적, 제약조건 설명
3. ✅ **구체적 요청**: 원하는 형식, 분량, 스타일 명시
4. ✅ **예시 제공**: 원하는 결과물의 예시 포함

<aside class="info">
<strong>프롬프트 팁</strong>: AI는 여러분의 의도를 완벽히 이해하지 못합니다. 마치 처음 만난 사람에게 설명하듯 구체적으로 작성하세요.
</aside>

### AI 출력물의 한계

생성형 AI는 강력하지만 한계가 있습니다:

- ⚠️ **사실 확인 필요**: AI가 만든 정보가 항상 정확하지는 않습니다
- ⚠️ **창의성 부족**: AI는 기존 데이터를 조합할 뿐, 진정한 창의성은 사람의 몫
- ⚠️ **맥락 이해 제한**: 복잡한 상황이나 뉘앙스를 완벽히 이해하지 못함
- ⚠️ **윤리적 검증**: 편향이나 부적절한 내용이 포함될 수 있음

**핵심**: AI는 **도구**입니다. 최종 판단과 책임은 사용자에게 있습니다.

## 실습 1: 시장 분석 및 문제점 도출

Duration: 20

### 시나리오

"생성형 AI 기반의 저비용 매장 음악 제공 서비스" 아이디어가 떠올랐습니다. 하지만 이것이 정말 시장성이 있는 아이디어인지 확인이 필요합니다.

### 실습 목표

생성형 AI를 활용하여:
1. 현재 매장 음악 시장의 문제점 파악
2. 타겟 고객 정의
3. 우리 서비스가 해결할 수 있는 핵심 문제 도출

### Step 1: 시장 조사 프롬프트 작성

ChatGPT 또는 Gemini에 다음 프롬프트를 입력하세요:

```
당신은 시장 조사 전문가입니다.

한국의 소규모 매장(카페, 식당, 소매점 등)에서 사용하는 
배경 음악 서비스 시장을 분석해주세요.

다음 내용을 포함해주세요:
1. 현재 시장 규모와 주요 서비스 (3가지)
2. 소상공인들이 겪는 주요 문제점 (5가지)
3. 각 문제점이 발생하는 이유
4. 시장 기회 (2-3가지)

각 항목은 명확하고 구체적으로 작성해주세요.
```

#### 주요 문제점

### Step 3: 추가 질문으로 깊이 파기

AI의 답변 중 궁금한 부분이 있다면 추가 질문을 하세요:

```
"저작권 문제"에 대해 더 자세히 설명해주세요. 
소상공인이 실제로 어떤 어려움을 겪나요?
```

또는

```
위에서 언급한 "음악 저작권료"의 실제 비용은 얼마나 되나요?
한국의 소규모 카페 기준으로 구체적인 금액을 알려주세요.
```

<aside class="info">
<strong>팁</strong>: AI와의 대화는 한 번에 끝나지 않습니다. 초기 답변을 바탕으로 계속 질문하며 정보를 구체화하세요. 이것이 프롬프트 엔지니어링의 핵심입니다.
</aside>

### Step 4: 비판적 검증

AI가 제공한 정보를 그대로 믿지 말고, 다음을 확인하세요:

1. ✅ **사실 확인**: 주요 숫자나 통계는 인터넷 검색으로 재확인
2. ✅ **최신성**: 정보가 최신인지 확인 (2024-2025년 기준)
3. ✅ **한국 상황**: 한국 시장에 맞는 정보인지 확인

의심스러운 부분은 다음과 같이 재질문하세요:

```
위에서 언급한 시장 규모 "XX억원"의 출처를 알려주세요.
만약 정확한 출처가 없다면, 어떻게 추정한 것인지 설명해주세요.
```

## 실습 2: 서비스 핵심 기능 정의

Duration: 25

### 실습 목표

앞서 파악한 문제점을 해결하는 **구체적인 서비스 기능**을 생성형 AI와 함께 설계합니다.

### Step 1: 솔루션 설계 프롬프트

```
당신은 IT 서비스 기획자입니다.

"생성형 AI 기반의 저비용 매장 음악 제공 서비스"를 기획하고 있습니다.

이전 분석에서 파악한 문제점:
1. 높은 음악 저작권료 (월 5-10만원)
2. 매장 분위기에 맞는 음악 선곡의 어려움
3. 음악 관리의 번거로움
4. 제한적인 음악 라이브러리
5. 시간대별 음악 변경의 어려움

위 문제점을 해결하는 서비스를 설계해주세요:

1. 서비스 이름 제안 (3가지)
2. 핵심 기능 5가지 (각 기능이 어떤 문제를 해결하는지 명시)
3. 차별화 포인트 (기존 서비스와 어떻게 다른가?)
4. 기술 구현 방식 (생성형 AI를 어떻게 활용할 것인가?)

실제 구현 가능한 수준으로 현실적으로 작성해주세요.
```

### Step 2: 프롬프트 개선 실습

첫 번째 결과가 마음에 들지 않는다면? 프롬프트를 개선하세요!

#### 개선 방법 1: 더 구체적으로

```
특히 "생성형 AI 활용" 부분을 더 구체적으로 설명해주세요.
어떤 AI 모델을 사용하고, 어떤 데이터로 학습하며, 
어떻게 저작권 문제를 해결할 것인지 포함해주세요.
```

#### 개선 방법 2: 제약조건 추가

```
서비스 기능을 다시 제안해주세요. 단, 다음 제약조건을 고려해주세요:
- 예산: 초기 투자 1억원 이하
- 개발 기간: 6개월 이내
- 개발 인력: 3명 (개발자 2명, 디자이너 1명)
- 타겟: 소상공인 (비용 민감)
```

#### 개선 방법 3: 역할 강화

```
당신은 실제로 음악 스타트업을 성공시킨 경험이 있는 
시리얼 앙트러프러너(serial entrepreneur)입니다.

실전 경험을 바탕으로, 위 서비스 기능 중에서 
"반드시 포함해야 할 기능"과 "나중에 추가해도 되는 기능"을 
구분해주세요. 그 이유도 함께 설명해주세요.
```

## 실습 3: 사업계획서 초안 작성

Duration: 30

### 실습 목표

지금까지 정리한 내용을 바탕으로 **투자자 또는 창업 경진대회에 제출할 수 있는 수준의 사업계획서**를 생성합니다.

### Step 1: 사업계획서 생성 프롬프트

```
당신은 스타트업 투자심사역(VC)입니다. 
수백 개의 사업계획서를 검토한 경험이 있습니다.

"생성형 AI 기반의 저비용 매장 음악 제공 서비스"에 대한 
사업계획서를 작성해주세요.

### 배경 정보:
- 서비스명: [앞서 정한 이름]
- 타겟: 한국의 소규모 매장 (카페, 식당, 소매점)
- 핵심 기능: [앞서 정의한 3-5가지 기능]
- 차별화: [앞서 정의한 차별화 포인트]

### 작성 요청 사항:
다음 구조로 A4 2페이지 분량의 사업계획서를 작성해주세요:

1. 사업 개요 (3-4문장)
2. 시장 기회
   - 타겟 시장 규모
   - 성장 가능성
3. 핵심 서비스
   - 주요 기능 3가지
   - 사용자 경험(UX) 흐름
4. 비즈니스 모델
   - 수익 구조
   - 가격 정책
5. 경쟁 우위
   - 기존 경쟁사 대비 차별점
6. 실행 계획
   - 6개월 로드맵
   - 필요 자원
7. 재무 계획 (간단히)
   - 초기 투자 필요 금액
   - 손익분기점 예상 시기

각 항목은 구체적이고 현실적으로 작성해주세요.
투자자가 읽고 바로 이해할 수 있도록 명확하게 작성해주세요.
```

### Step 2: 출력물 검토 체크리스트

AI가 생성한 사업계획서를 다음 기준으로 검토하세요:

#### 내용 검토
- [ ] 모든 섹션이 포함되어 있는가?
- [ ] 숫자와 데이터가 구체적인가?
- [ ] 주장에 대한 근거가 명확한가?
- [ ] 실현 가능성이 있는가?

#### 문장 검토
- [ ] 문장이 명확하고 이해하기 쉬운가?
- [ ] 전문 용어가 과도하지 않은가?
- [ ] 한국어 표현이 자연스러운가?
- [ ] 맞춤법과 띄어쓰기가 정확한가?

#### 설득력 검토
- [ ] 투자자가 관심을 가질 만한가?
- [ ] 차별화 포인트가 명확한가?
- [ ] 실행 계획이 현실적인가?

### Step 3: 나만의 스타일로 수정하기

AI가 생성한 문서를 **그대로 사용하지 마세요**. 다음과 같이 개인화하세요:

#### 1. 톤 앤 매너 조정

원본 (AI 생성):
```
본 서비스는 혁신적인 AI 기술을 통해 음악 산업의 
패러다임을 전환하고자 합니다.
```

수정 (더 솔직하고 현실적으로):
```
소상공인들이 매달 내는 음악 저작권료, 부담스럽죠? 
우리는 AI로 생성한 음악으로 이 문제를 해결합니다.
```

#### 2. 구체적인 스토리 추가

AI는 일반적인 내용만 생성합니다. 여러분의 경험이나 인터뷰를 추가하세요:

```
"강남역 인근 카페를 운영하는 김OO 사장님은 말했습니다.
'매달 8만원씩 나가는 음악 저작권료가 부담이에요. 
하지만 음악 없는 카페는 상상할 수 없잖아요.'"
```

#### 3. 비주얼 요소 추가

- 서비스 구조를 간단한 다이어그램으로
- 주요 숫자는 시각적으로 강조
- 사용자 여정(User Journey)을 그림으로 표현

### Step 4: 최종 문서 완성

Google Docs에 다음 형식으로 최종 정리:

1. **표지**: 서비스명, 팀명, 날짜
2. **본문**: AI 생성 + 여러분의 수정
3. **부록**: 추가 자료, 참고 링크

<aside class="info">
<strong>중요</strong>: 이 사업계획서는 "AI로 생성했다"는 사실을 명시할 필요는 없지만, 내용에 대한 책임은 여러분에게 있습니다. 모든 정보를 검증하세요.
</aside>

## 실습 4: 프롬프트 개선 실전

Duration: 20

### 실습 목표

앞서 만든 사업계획서를 **더욱 개선**하기 위한 프롬프트 엔지니어링 실전 연습을 합니다.

### 시나리오: 부족한 부분 보완하기

사업계획서를 검토한 결과, "경쟁 분석" 부분이 약하다는 피드백을 받았습니다.

### 방법 1: 직접 보완 요청

```
위 사업계획서의 "경쟁 우위" 섹션을 더 강화해주세요.

다음 내용을 추가해주세요:
1. 주요 경쟁사 3곳 (한국 시장 기준)
   - 각 경쟁사의 서비스 특징
   - 가격 정책
   - 장단점
2. 우리 서비스와의 비교표
3. 우리가 이길 수 있는 이유 (구체적인 근거와 함께)

표 형식으로 정리하고, 객관적으로 작성해주세요.
```

### 방법 2: 역할극 활용

```
당신은 우리 서비스의 가상 경쟁자입니다.
당신의 임무는 우리 서비스의 약점을 찾아 공격하는 것입니다.

우리 서비스 개요:
[앞서 작성한 내용 붙여넣기]

다음을 작성해주세요:
1. 우리 서비스의 5가지 약점
2. 각 약점을 공략하는 전략
3. 우리가 실패할 수 있는 시나리오

가능한 냉정하고 비판적으로 분석해주세요.
```

AI의 답변을 받은 후, 이를 바탕으로 보완책을 마련하세요.

### 방법 3: 멀티 프롬프트 기법

하나의 긴 프롬프트 대신, 여러 개의 짧은 프롬프트로 나누어 질문하세요:

**첫 번째 프롬프트:**
```
한국의 매장 음악 서비스 시장의 주요 경쟁사를 5곳 나열해주세요.
각 회사명과 서비스명을 표로 정리해주세요.
```

**두 번째 프롬프트:**
```
위에서 언급한 5개 서비스 중, 
"멜론 비즈니스"와 "네이버 VIBE 비즈"에 대해 
더 자세히 분석해주세요:
- 제공 음악 수
- 월 이용료
- 주요 기능
- 사용자 리뷰
```

**세 번째 프롬프트:**
```
이제 우리 서비스와 비교해주세요.
우리의 차별화 포인트는 다음과 같습니다:
[내용 입력]

비교표를 만들어주세요.
```

### Step 4: A/B 테스트

동일한 질문을 다르게 표현하여 여러 번 물어보고, 가장 좋은 답변을 선택하세요:

**버전 A:**
```
우리 서비스의 수익 모델을 제안해주세요.
```

**버전 B:**
```
당신은 성공한 SaaS 스타트업 CEO입니다.
우리 서비스(매장 음악 제공)의 수익 모델을 3가지 제안하고,
각 모델의 장단점과 예상 매출을 비교해주세요.
```

어느 프롬프트가 더 유용한 답변을 생성하나요?

<aside class="info">
<strong>프롬프트 팁</strong>: 
- 짧고 간단한 프롬프트 → 빠른 답변, 하지만 일반적인 내용
- 길고 구체적인 프롬프트 → 시간이 걸리지만, 더 맞춤화된 답변
상황에 따라 전략을 선택하세요!
</aside>

## 발표 및 토론

Duration: 15

### 발표 준비

각자(또는 팀별로) 완성한 사업계획서를 **5분 이내**로 발표할 준비를 하세요.

### 발표 구조 제안

1. **도입 (1분)**: 문제 제기
   - "이런 문제를 발견했습니다"
   
2. **솔루션 (2분)**: 우리의 서비스
   - 핵심 기능 3가지
   - 차별화 포인트
   
3. **실행 (1분)**: 어떻게 만들 것인가
   - 기술 스택
   - 일정
   
4. **마무리 (1분)**: 왜 성공할 것인가
   - 시장 기회
   - 우리의 강점

### 발표 시 공유할 내용

- **프롬프트 경험**: 어떤 프롬프트가 가장 효과적이었나요?
- **AI의 한계**: 어떤 부분에서 AI가 도움이 되지 않았나요?
- **놀라운 발견**: AI를 사용하면서 예상치 못한 인사이트가 있었나요?

### 토론 주제

#### 1. 프롬프트 설계의 어려움
- 가장 어려웠던 점은 무엇인가요?
- 어떻게 극복했나요?

#### 2. AI 출력물의 검증
- AI의 답변 중 잘못된 정보를 발견했나요?
- 어떻게 검증했나요?

#### 3. 생성형 AI의 실무 활용
- 취업 준비(자기소개서, 이력서)에 어떻게 활용할 수 있을까요?
- 실무에서 생성형 AI가 가장 유용할 분야는?

#### 4. 윤리적 고민
- AI로 생성한 내용을 "내 것"이라고 할 수 있을까요?
- 어디까지가 허용되고, 어디서부터 문제일까요?

<aside class="info">
<strong>발표 팁</strong>: 사업계획서 전체를 읽지 마세요. 핵심 3가지만 명확히 전달하는 것이 훨씬 효과적입니다.
</aside>

## 정리 및 과제

Duration: 10

### 핵심 요약

이번 강의에서 배운 내용:

1. ✅ **생성형 AI는 도구다**: AI는 초안을 만들고, 아이디어를 확장하는 데 유용합니다. 하지만 최종 판단은 여러분의 몫입니다.

2. ✅ **좋은 프롬프트 = 좋은 결과**: 프롬프트 엔지니어링은 연습이 필요합니다. 계속 시도하고 개선하세요.

3. ✅ **비판적 검증 필수**: AI의 출력물을 무조건 신뢰하지 마세요. 항상 검증하고, 사실을 확인하세요.

4. ✅ **나만의 스타일로 가공**: AI 출력물을 그대로 사용하지 말고, 여러분의 경험과 관점을 더하세요.

### 생성형 AI 활용 체크리스트

앞으로 생성형 AI를 사용할 때 이 체크리스트를 활용하세요:

**작업 전**
- [ ] 명확한 목표가 있는가?
- [ ] 충분한 배경 정보를 제공할 준비가 되었는가?
- [ ] 원하는 결과물의 형식을 정의했는가?

**작업 중**
- [ ] 프롬프트가 구체적이고 명확한가?
- [ ] 역할, 맥락, 요청사항을 포함했는가?
- [ ] 만족스럽지 않으면 프롬프트를 수정하고 다시 시도했는가?

**작업 후**
- [ ] AI의 답변을 비판적으로 검토했는가?
- [ ] 주요 정보를 사실 확인했는가?
- [ ] 나만의 스타일로 수정했는가?
- [ ] 출처를 명확히 했는가?

### 과제

다음 수업까지 완료해야 할 과제:

#### 필수 과제
개인별로 **자신의 전공/희망 직무와 관련된 기획서**를 생성형 AI를 활용하여 작성하세요.

**제출 내용:**
1. 완성된 기획서 (A4 1-2페이지)
2. 사용한 주요 프롬프트 3-5개
3. 프롬프트를 어떻게 개선했는지 설명 (200-300자)
4. AI 출력물 중 수정한 부분과 이유 (200-300자)

**형식**: Google Docs 또는 PDF로 제출

#### 선택 과제 (추가 점수)
다음 중 하나를 선택하여 작성하세요:

1. **프롬프트 라이브러리**: 내가 유용하게 사용한 프롬프트 10개를 모아 정리
2. **AI 비교 리포트**: ChatGPT, Gemini, Claude 등 3가지 AI 도구를 사용해보고 장단점 비교
3. **실패 사례집**: AI가 잘못된 정보를 제공한 사례 3가지와 어떻게 발견/수정했는지

## 부록: 프롬프트 템플릿 모음

Duration: 5

### 사업 기획 관련

#### 시장 조사 템플릿
```
당신은 시장 조사 전문가입니다.

[산업/분야]의 [특정 시장]을 분석해주세요.

포함 내용:
- 시장 규모와 성장률
- 주요 플레이어 3-5곳
- 트렌드 및 기회
- 위험 요소

[국가/지역] 시장 기준으로 작성해주세요.
```

#### 경쟁 분석 템플릿
```
다음 서비스/제품의 경쟁 분석을 해주세요:
- 내 서비스: [설명]
- 타겟: [고객군]

주요 경쟁사 [N]개를 찾아, 다음을 표로 정리해주세요:
- 회사명/서비스명
- 핵심 기능
- 가격
- 강점/약점
- 시장 점유율 (추정)
```

#### 비즈니스 모델 템플릿
```
당신은 린 스타트업 전문가입니다.

다음 서비스의 비즈니스 모델을 설계해주세요:
- 서비스: [설명]
- 타겟: [고객군]
- 문제: [해결하려는 문제]

다음 형식으로 작성해주세요:
1. 가치 제안 (Value Proposition)
2. 고객 세그먼트
3. 수익 모델 (3가지 옵션)
4. 비용 구조
5. 핵심 지표 (KPI)
```

요약 형식:
- 한 줄 요약 (20자 이내)
